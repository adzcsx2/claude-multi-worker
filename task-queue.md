# Task Queue - Multi-Window Collaboration

## Window Roles

### Required Roles
- **Window 1: Main Task (Dev)** - Architecture, Development, Bug Fixes
- **Window 2: Test Task (QA)** - Unit Tests, UI Simulation, Quality Checks

### Recommended Roles
- **Window 3: Design Task (UI/UX)** - UI Design Analysis, Interface Implementation
- **Window 4: Documentation Task** - API Docs, User Manuals, README

### Optional Roles
- **Window 5: Review Task (Code Review)** - Code Review, Security Audit
