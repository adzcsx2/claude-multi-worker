===============================================
✅ 已修复为标签页模式
===============================================

推荐命令:
  python cms.py claude:ui,coder,test

或:
  python cms.py claude

===============================================
工作原理:
===============================================

1. 检测是否在 WezTerm 中
2. 发送 Ctrl+Shift+T 创建新标签页
3. 在每个标签页中启动 Claude
4. 设置标签页标题

===============================================
使用步骤:
===============================================

1. 打开 WezTerm
   cd E:\ai_project\claude-multi-starter

2. 启动多标签页
   python cms.py claude:ui,coder,test

3. 使用 Ctrl+Tab 切换标签页

4. 在另一个终端发送消息
   bin\send ui "设计任务"
   bin\send coder "开发任务"
   bin\send test "测试任务"
