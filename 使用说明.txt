===============================================
  CMS 使用说明 - 重要!
===============================================

❌ 错误用法:
--------------
python cms.bat tab    # 错误! .bat文件不能用python执行


✅ 正确用法:
--------------

Windows CMD:
  cms.bat tab
  cms.bat tab ui,coder,test
  cms.bat pane ui,coder
  cms.bat window ui,coder,test

Windows PowerShell / Linux / Mac:
  python cms.py tab
  python cms.py tab ui,coder,test
  python cms.py pane ui,coder
  python cms.py window ui,coder,test

直接调用:
  python START_MULTI_TAB.py ui,coder,test
  python START_MULTI_PANE.py ui,coder,test
  python START_MULTI_WINDOW.py ui,coder,test


===============================================
  快速开始
===============================================

1. 查看帮助
   python cms.py

2. 启动标签页模式 (推荐)
   python cms.py tab

3. 发送消息
   bin\send ui "设计任务"
   bin\send coder "开发任务"
   bin\send test "测试任务"


===============================================
  完整示例
===============================================

# 步骤1: 启动多个Claude实例 (标签页模式)
python cms.py tab ui,coder,test

# 步骤2: 在另一个终端窗口发送消息
bin\send ui "设计一个登录页面,包含用户名和密码输入框"
bin\send coder "实现登录API,使用JWT认证"
bin\send test "为登录功能编写测试用例"

# 步骤3: 在WezTerm中切换标签页查看结果
# Ctrl+Tab - 切换到下一个标签页
# Ctrl+Shift+Tab - 切换到上一个标签页


===============================================
  三种模式对比
===============================================

标签页模式 (推荐):
  命令: python cms.py tab
  特点: 在一个窗口中,多个标签页
  优点: 紧凑,快速切换 (Ctrl+Tab)
  
窗格模式:
  命令: python cms.py pane
  特点: 在一个窗口中,分割多个窗格
  优点: 可同时看到多个实例
  
多窗口模式:
  命令: python cms.py window
  特点: 每个实例独立窗口
  优点: 适合多显示器
